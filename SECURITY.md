# Security Policy

## Supported Versions

Security fixes are provided for the latest stable release of Marwa DB.

## Reporting a Vulnerability

If you believe you have found a security issue, please do not open a public issue.

Instead, report it privately to the project maintainers using the contact details in the repository owner profile or the project documentation.

Include as much detail as possible:

- A description of the issue
- Affected version(s)
- Steps to reproduce
- Any error messages, stack traces, or proof of concept code

We will review reports promptly and respond as soon as practical.
